非官方 Windows 与 macOS 补丁安装器

Codex Patched 安装器

在你的电脑上基于已安装的官方 Codex App 构建一个独立的补丁副本。 安装包不包含、也不重新分发官方 Codex App 二进制文件。

下载 macOS ZIP 下载 Windows ZIP 查看源码
macOS 安装 解压 ZIP 后运行 installer/macos/Install-CodexPatched-macos.sh。
Windows 安装 在 PowerShell 中运行 .\Install-CodexPatched.ps1。
共同要求 已安装官方 Codex 桌面应用，以及带 npx 的 Node.js LTS。
